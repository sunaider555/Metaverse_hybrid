Metaverse Wallet

Welcome to Metaverse Wallet, your all-in-one Metaverse node and wallet.

WARNING: This installer is **EXPERIMENTAL**. Use it at your own risk.

If you continue, Metaverse will be installed as a user service. You will be able to use the Metaverse Wallet through your browser by using the menu bar icon, following the shortcut in the Launchpad or navigating to http://localhost:8820/ in your browser.
